## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Groq API (LLaMA 3.3 70B)
- LocalStorage for data persistence

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Start chatting!

## Privacy

All your conversation data is stored locally in your browser's localStorage. Nothing is sent to external servers except the chat messages to the AI API.

## Disclaimer

MindfulChat is a supportive companion, not a replacement for professional mental health care. If you're experiencing a mental health crisis, please contact emergency services or a mental health professional.
